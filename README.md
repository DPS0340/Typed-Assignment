## How to use?

### Vercel 배포 환경 사용

[Vercel](https://typed-assignment.vercel.app/)에 배포되어 있는 데모를 확인하실 수 있습니다.


### 로컬 개발 환경 구성

먼저 해당 레포에서 패키지 매니저로 사용하고 있는 [pnpm](https://pnpm.io/)을 설치한 뒤, 의존성을 가져옵니다.

```bash
npm i -g pnpm
pnpm i
```

Vite dev 서버를 실행합니다.

```bash
pnpm run dev
```
